# ⚡ 電池壽命監控看板 (Battery Life Monitoring SCADA)

這是一個極致黑 SCADA 風格的智慧儲能機櫃全局監控戰情室。本系統基於機器學習模型與電池退化物理特徵，提供實時的電池健康度 (SOH) 評估與剩餘使用壽命 (RUL) 預測。

## 🌟 主要功能特色

*   **🎛️ SCADA 沉浸式戰情面板**：採用全局黑化、高對比度的 UI 設計，模擬真實工業控制系統介面。隱藏了預設的 Streamlit 選單，提供全螢幕的專業視覺體驗。
*   **🤖 AI 壽命預測引擎**：整合多種機器學習演算法 (整合於 `probms_model.pkl`)，包含 Isotonic Regression 確保退化軌跡呈現物理單調性，精準預估剩餘循環次數 (RUL)。
*   **📊 即時物理特徵解析**：
    *   **健康雷達圖**：透過五大指標 (容量保持、低內阻、極化穩定、壓降健康、壽命預估) 全方位診斷電池陣列。
    *   **退化軌跡圖**：可視化對比「實際 RUL」與「AI 預測路徑」，一目了然電池的衰退趨勢。
*   **⏳ 動態時間軸與自動模擬**：
    *   支援手動拖曳時間軸 (Cycle) 檢視歷史與未來預測狀態。
    *   支援 **自動播放 (Auto-Play)**，即時動態模擬，並能根據「每日循環強度」自動換算**相當於運轉了幾天**，讓壽命流失更有感。
*   **🚨 智能警報與專家日誌**：
    *   依據 SOH 自動切換機櫃狀態 (正常 ONLINE / 注意 WARNING / 停機 CRITICAL)。
    *   終端機風格 (Terminal-style) 的稽核日誌，記錄放電異常與即時診斷結果。

## 🛠️ 系統需求與安裝

請確保您的環境已安裝 Python 3.8 或以上版本。

1.  **安裝依賴套件**：
    使用 `pip` 安裝 `requirements.txt` 中條列的必要套件：
    ```bash
    pip install -r requirements.txt
    ```
    *(核心依賴包含: streamlit, pandas, numpy, plotly, joblib, scikit-learn, streamlit_autorefresh 等)*

2.  **準備資料與模型**：
    確保應用程式根目錄下包含以下必要文件：
    *   `Battery_RUL.csv` - 電池運行數據與特徵資料。
    *   `probms_model.pkl` - 預先訓練好的機器學習模型包。

## 🚀 執行應用程式

在終端機 (Terminal) 中，進入專案目錄並執行以下指令啟動 Streamlit 伺服器：

```bash
streamlit run app.py
```

啟動後，瀏覽器將自動開啟應用程式 (預設網址通常為 `http://localhost:8501`)。

## 📖 介面操作指南

1.  **控制台 (側邊欄)**：
    *   **分析機櫃單元**：選擇要監控的特定電池陣列 ID。
    *   **每日循環強度**：設定該機櫃每天操作的循環(Cycle)次數，系統會依此將預測剩餘次數換算成「可用年資」與「已運轉天數」。
    *   **時間軸模擬 / 自動播放**：可用拉桿手動查看特定時間點或是開啟自動播放進行自動化模擬展演。
2.  **核心指標 (KPIs)**：
    頂部四大數據塊分別呈現：最新預測剩餘次數 (RUL)、預估剩餘可用年數、當前健康評分 (SOH)、以及系統最終狀態判定。
3.  **圖表區與日誌區**：
    畫面中央及下方提供視覺化圖表輔助研判，底部「專家系統診斷與稽核日誌」則會輸出系統的事件與狀態紀錄。
